# INF1009 Abstract Engine (Part 1)

A small **UML-aligned abstract 2D engine** for INF1009 OOP, plus a **minimal demo** to showcase the engine systems.

## What this is
- **Part 1 = Abstract Engine only**
- Generic and reusable (not tied to any specific game)
- Demo screens exist only to prove the engine works
